How to run the Hospital Management System Project Using PHP and MySQL

1. Download the zip file
2. Extract the file and copy folder
3. Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name hospital
6. Import hospital.sql file(given inside the zip package in SQL file folder)
7. Run the script http://localhost/hms/Hospital-Management-System/hospital/MYPROJECT/(frontend)
